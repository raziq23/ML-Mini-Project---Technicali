# MCTA4362 - Intelligent Temperature Control
## Machine Learning Mini Project

### System
- Plant: First-Order Thermal System
- Setpoint: 75°C

### Controllers
- Classical: PID Controller
- ML-Based: Artificial Neural Network (Supervised Learning)

### Files
| File | Description |
|------|-------------|
| thermal_model.m | Thermal plant modelling |
| pid_temperature.m | PID controller design |
| pid_metrics.m | PID performance metrics |
| pid_disturbance.m | PID disturbance test |
| ann_data_collection.m | ANN training data collection |
| ann_train.m | ANN training and validation |
| ann_vs_pid.m | PID vs ANN comparison |

### Requirements
- MATLAB R2021a or later
- Deep Learning Toolbox
- Control System Toolbox

### Run Order
1. thermal_model.m
2. pid_temperature.m
3. pid_metrics.m
4. pid_disturbance.m
5. ann_data_collection.m
6. ann_train.m
7. ann_vs_pid.m
